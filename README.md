# 抖视频

仿抖音竖屏滑动浏览播放视频 App，基于 Flutter 开发，支持 Android 和 Windows 平台。

## 功能特性

- 🎬 竖屏全屏视频播放
- 👆 上下滑动切换视频
- ❤️ 点赞、评论、收藏、分享交互
- 📂 视频分类管理 (创建/编辑/删除分类)
- ✏️ 视频信息编辑 (重命名标题、编辑描述)
- 📋 批量操作 (批量删除、批量归类)
- 🎵 播放列表 (创建/编辑/删除播放列表)
- ⏰ 定时关闭播放 (15/30/60 分钟/自定义)
- 📊 播放统计 (播放次数、观看时长、最后播放时间)
- 🔄 自动连播开关
- ⚙️ 设置页面 (清除历史、清除缓存)
- 🌓 明暗主题支持
- 📱 Android 手机 + 折叠屏 + 平板自适应
- 🖥️ Windows 10+ 窗口自适应
- 🌐 简体中文界面

## 技术栈

- **框架**: Flutter 3.44+
- **状态管理**: Provider
- **视频播放**: media_kit
- **存储**: shared_preferences
- **桌面窗口**: window_manager
